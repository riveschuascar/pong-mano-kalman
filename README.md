# Requerimientos
* Python > 3.13.0
* mediapipe
* pygame

# Antes de ejecutar
* Crear entorno vitual
```py -3.12 -m venv .env```
* Activar entorno virtual
Mac/Linux:
```source .env/bin/activate```
Windows"
```.env/Scripts/activate.bat```
* Instalar packetes de dependencias
```pip install -r requirements.txt```

# Ejecutar
```py scr/main.py```
